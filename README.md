## Instalasi

1. Clone Repository

```sh
git clone https://github.com/zuramai/refactoring
cd refactoring
```

2. Install dependencies

```sh
pnpm install
# ATAU npm install
# ATAU yarn install
```

3. Jalankan

```sh
pnpm run docs:dev
```

4. Buka `http://localhost:3000` di browser
